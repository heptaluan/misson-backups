斐盼安计算公式：
I = 影像得分
L = I > 60 ? (I - 60)* 3 / 4 + 70 : I * 7 / 6; 
Z = R（综合得分）* 1.2 - L * 0.2

斐盼康计算公式：
I = 影像得分 
L = I > 60 ? (I - 60)* 3 / 4 + 70 : I * 7 / 6;  
S = 表观得分
B = S > 60 ? (S - 60)* 3 / 4 + 70 : S * 7 / 6;  
Z = R（综合得分）* 1.4 - L * 0.2 - B * 0.2
